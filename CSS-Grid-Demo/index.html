Certain Color Themes in VS Code

The red color theme in vs code is fun to look until you stare at it for too long.

It gives you a headache while making you drowsy.

The abyss theme doesn't make you want to fall asleep but it has similar feels to the red theme

One is really red and the other is very blue.

The Red Theme

The red theme is extremely red, to the point that you cannot read certain texts

The warm color scheme also makes me want to look at it and fall asleep

The Abyss Theme

the abyss does a better a job of taking away drowsiness but it is also quite hard to see certain texts.

The background, the color of the text and the highlited line causes the root of these problems.

So, my suggestion would be to use it only if you are great at distingruishing between colors and it does not distract you.

Because, I know that I definitely got distracted.
