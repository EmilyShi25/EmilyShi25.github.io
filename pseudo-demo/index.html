What are Pseudo Classes?

Again, if we think of CSS in terms of selectors, properties and values, then a pseudo class is an add-on to a selector 

            selector: pseudo_class {property: value;}

We use them for links

if we want our Link to look different, we can simply add the following

            a {color: red;}

But if we want each of our link states to look different then we weill use a pseudo class like the following

  • :hover
  • :active
  • visited
